# 🌰 Crypto Threat Intelligence Analyzer

An AI-powered threat intelligence platform that monitors and analyzes cryptocurrency security threats in real-time, generating weekly intelligence reports for the blockchain community.

## 🚀 Features

- **AI-Powered Analysis**: Leverages GitHub Models (GPT-4o) for intelligent threat pattern analysis
- **Multi-Chain Monitoring**: Tracks threats across Ethereum, BSC, Polygon, Arbitrum, and more
- **Weekly Reports**: Automated generation of comprehensive threat intelligence reports
- **Risk Assessment**: Severity-based threat categorization and impact analysis
- **Actionable Recommendations**: Provides specific security recommendations for users and developers

## 🛠️ How It Works

1. **Data Collection**: Aggregates security incidents from various crypto news sources
2. **AI Analysis**: Uses advanced language models to identify patterns and trends
3. **Report Generation**: Creates beautiful HTML reports with interactive visualizations
4. **Automated Deployment**: Deploys reports to GitHub Pages every Monday

## 📊 Report Contents

- Threat summary statistics
- Detailed incident analysis
- Risk assessment by blockchain ecosystem
- AI-generated predictions for upcoming week
- Security recommendations and best practices

## 🔧 Setup

1. Fork this repository
2. Enable GitHub Pages in repository settings
3. Ensure GitHub Actions has necessary permissions
4. The analyzer runs automatically every Monday at 9 AM UTC
