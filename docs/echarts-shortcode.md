# 🌰 Dynamic Charts with `echarts` Shortcode

The `echarts` Hugo shortcode renders interactive [Apache ECharts 5](https://echarts.apache.org/)
charts directly in Markdown articles. It replaces static PNG images generated by matplotlib — with
**zero runtime dependencies** on matplotlib, pandas, or numpy.

## Why ECharts? 🌰

| Feature | Chart.js | **ECharts 5 (this PR)** |
|---|---|---|
| Native financial charts (candlestick, OHLC) | ❌ | ✅ |
| Built-in zoom / pan | ❌ plugin needed | ✅ `dataZoom` |
| Save-to-PNG toolbox | ❌ | ✅ |
| Colour gradient (`visualMap`) | ❌ | ✅ green → yellow → red |
| Dark mode | via CSS | ✅ `echarts.init(dom, 'dark')` |
| Dual y-axis | manual config | ✅ native `yAxisIndex` |
| Bundle size overhead | ~200 KB (bundled) | ~1 MB (CDN, loaded once/page) |

## Quick Start

```markdown
{{< echarts id="vol-hist" title="🌰 Volume Distribution" height="350" >}}
{
  "xAxis": {"type": "category", "data": ["0-10", "10-20", "20-30"]},
  "yAxis": {"type": "value"},
  "series": [{"type": "bar", "data": [120, 95, 48]}]
}
{{< /echarts >}}
```

## Parameters

| Parameter | Default | Description |
|-----------|---------|-------------|
| `id` | auto (timestamp) | Unique container element ID |
| `title` | — | Chart title (rendered inside ECharts, not as HTML heading) |
| `height` | `400` | Container height in pixels |
| `caption` | — | Figure caption rendered below the chart |

## Inner Content (JSON)

The shortcode body must be a valid JSON object matching the
[ECharts `option` schema](https://echarts.apache.org/en/option.html).

The shortcode **deep-merges** your JSON with sensible defaults
(tooltip, legend, dataZoom, toolbox, responsive grid) — you only need to
supply the parts you want to override.

```json
{
  "xAxis": { "type": "category", "data": ["Jan", "Feb", "Mar"] },
  "yAxis": { "type": "value" },
  "series": [{ "type": "bar", "name": "Volume", "data": [3200, 4100, 2900] }]
}
```

## Chart Examples for Market Health Reports 🌰

### 1. Volume Histogram

```markdown
{{< echarts id="volume-hist" title="🌰 Transaction Volume Distribution" caption="30-bin histogram of transaction volumes" >}}
{
  "xAxis": {"type":"category","data":["0.00","33.33","66.67"],"name":"Transaction Volume","nameLocation":"middle","nameGap":35,"axisLabel":{"rotate":30}},
  "yAxis": {"type":"value","name":"Frequency"},
  "series": [{"type":"bar","data":[150,89,42],"itemStyle":{"color":"rgba(135,206,235,0.85)","borderColor":"#000","borderWidth":0.5}}]
}
{{< /echarts >}}
```

### 2. Time-Series (Volume / Trade Count / Avg Tx Size / Buy-Sell Ratio)

```markdown
{{< echarts id="volume-ts" title="🌰 Volume Over Time" height="300" >}}
{
  "xAxis": {"type":"category","data":["2023-07-01 00:00","2023-07-01 01:00"],"axisLabel":{"rotate":30}},
  "yAxis": {"type":"value","name":"Volume"},
  "series": [{"name":"Volume","type":"line","data":[1234.5,2345.6],"lineStyle":{"color":"#5470c6"},"itemStyle":{"color":"#5470c6"},"showSymbol":false,"smooth":true,"areaStyle":{"opacity":0.1}}]
}
{{< /echarts >}}
```

### 3. Benford Law (Dual Y-Axis) 🌰

Benford score (left axis) vs. critical value 1.36/√N (right axis, dashed).
Scores above the dashed line indicate statistically anomalous distributions.

```markdown
{{< echarts id="benford-law" title="🌰 Benford Law Test Score" caption="Scores above critical value (dashed) indicate anomalies" >}}
{
  "xAxis": {"type":"category","data":["2023-07-01","2023-07-02"],"axisLabel":{"rotate":30}},
  "yAxis": [
    {"type":"value","name":"Benford Score","position":"left"},
    {"type":"value","name":"Critical Value","position":"right","splitLine":{"show":false}}
  ],
  "series": [
    {"name":"Benford Law Score","type":"line","yAxisIndex":0,"data":[0.045,0.032],"lineStyle":{"color":"#5470c6"},"showSymbol":false,"smooth":true},
    {"name":"Critical Value (1.36/√N)","type":"line","yAxisIndex":1,"data":[0.028,0.030],"lineStyle":{"color":"#91cc75","type":"dashed"},"showSymbol":false}
  ]
}
{{< /echarts >}}
```

### 4. Volume-Volatility Correlation (visualMap gradient) 🌰

Uses ECharts `visualMap` to apply a **green → yellow → red** gradient based on correlation
magnitude — high-correlation periods stand out at a glance.

```markdown
{{< echarts id="vv-correlation" title="🌰 Volume-Volatility Correlation Over Time" caption="Gradient: green (low) → red (high correlation)" >}}
{
  "xAxis": {"type":"category","data":["2023-07-01","2023-07-02"],"axisLabel":{"rotate":30}},
  "yAxis": {"type":"value","name":"VV Correlation"},
  "visualMap": {
    "show":false,"type":"continuous","seriesIndex":0,
    "min":0.5,"max":0.95,
    "inRange":{"color":["#91cc75","#fac858","#ee6666"]}
  },
  "series": [{"name":"VV Correlation","type":"line","data":[0.72,0.88],"showSymbol":true,"symbolSize":4,"smooth":true}]
}
{{< /echarts >}}
```

### 5. Candlestick / OHLC (ECharts-exclusive) 🌰

ECharts natively supports candlestick charts — a format Chart.js cannot produce without plugins.

```markdown
{{< echarts id="ohlc" title="🌰 OHLC Price" height="450" >}}
{
  "xAxis": {"type":"category","data":["2023-07-01","2023-07-02","2023-07-03"]},
  "yAxis": {"type":"value","scale":true},
  "series": [{
    "type": "candlestick",
    "data": [
      [29500, 29800, 29200, 29750],
      [29750, 30100, 29600, 29900],
      [29900, 30500, 29800, 30300]
    ]
  }]
}
{{< /echarts >}}
```

## How the Pipeline Works 🌰

```
Market Health API (JSON)
        │
        ▼
LLM generates article text
        │
        ▼
Visualization.generate_report(data, directory)
   ├─ _make_volume_hist()      → bar histogram
   ├─ _make_crypto_metrics()   → 4× line charts
   ├─ _make_benfordlaw()       → dual-axis line
   └─ _make_vvcorrelation()    → visualMap gradient line
        │
        ▼
Shortcode JSON blocks appended to index.md
        │
        ▼
Hugo renders {{< echarts >}} → interactive ECharts canvas
```

## Features Summary

| Feature | Detail |
|---------|--------|
| Zero Python deps | No matplotlib, pandas, or numpy |
| Responsive | Auto-resizes on `window.resize` |
| Dark / light | Detects `prefers-color-scheme` + `dark` CSS class; re-inits chart on change |
| CDN once per page | `.Page.Scratch` guard (same pattern as `katex.html`) |
| Deep merge | User JSON overrides only what's needed — defaults handle the rest |
| Toolbox | Built-in zoom, restore, save-to-PNG 🌰 |
| dataZoom | Slider + inside scroll/pinch zoom out of the box |

## How the Shortcode Loads ECharts

ECharts (~1 MB minified) is loaded **once per page** from jsDelivr CDN:

```html
<script src="https://cdn.jsdelivr.net/npm/echarts@5/dist/echarts.min.js"></script>
```

The `.Page.Scratch` guard (same pattern as the existing `katex.html` shortcode) ensures only one
`<script>` tag is emitted even when multiple `{{< echarts >}}` shortcodes appear on the same page.

🌰🌰🌰 Chestnut overlords approve of this approach.
