# Dynamic Charts with `metric_chart` Shortcode

The `metric_chart` Hugo shortcode renders interactive [Chart.js](https://www.chartjs.org/) charts
directly in markdown articles. It replaces static PNG images generated by matplotlib.

## Quick Start

```markdown
{{</* metric_chart id="my-chart" type="bar" title="Volume Distribution" */>}}
{"labels":["0-10","10-20","20-30"],"datasets":[{"label":"Frequency","data":[120,95,48]}]}
{{</* /metric_chart */>}}
```

## Parameters

| Parameter | Default | Description |
|-----------|---------|-------------|
| `id`      | auto    | Unique canvas element ID |
| `type`    | `bar`   | Chart type: `bar`, `line`, `scatter` |
| `title`   | —       | Chart title text |
| `height`  | `400`   | Canvas height in pixels |
| `caption` | —       | Figure caption below the chart |

## Inner Content (JSON)

The shortcode body must be a valid JSON object:

```json
{
  "labels": ["label1", "label2"],
  "datasets": [
    {
      "label": "Series Name",
      "data": [10, 20],
      "borderColor": "rgba(54,162,235,1)",
      "type": "line"
    }
  ],
  "options": {}
}
```

The `options` key is merged with the shortcode defaults and accepts any
[Chart.js options](https://www.chartjs.org/docs/latest/configuration/).

## Chart Types for Market Health Reports

### 1. Volume Histogram

```markdown
{{</* metric_chart id="volume-hist" type="bar" title="Transaction Volume Distribution" */>}}
{"labels":["0.00","33.33","66.67","100.00"],"datasets":[{"label":"Frequency","data":[150,89,42,12],"backgroundColor":"rgba(135,206,235,0.7)","borderColor":"rgba(0,0,0,0.8)","borderWidth":1}]}
{{</* /metric_chart */>}}
```

### 2. Time-Series (Volume / Trade Count / Avg Tx Size / Buy-Sell Ratio)

```markdown
{{</* metric_chart id="metric-volume" type="line" title="Volume Over Time" height="280" */>}}
{"labels":["2023-07-01 00:00","2023-07-01 01:00","2023-07-01 02:00"],"datasets":[{"label":"Volume","data":[1234.5,2345.6,1890.2],"borderColor":"rgba(54,162,235,0.8)","fill":true,"pointRadius":0,"tension":0.2}]}
{{</* /metric_chart */>}}
```

### 3. Benford Law (Dual Y-Axis)

```markdown
{{</* metric_chart id="benford-law" type="line" title="Benford Law Test Score" caption="Scores above critical value indicate anomalies" */>}}
{"labels":["2023-07-01 00:00","2023-07-01 01:00"],"datasets":[{"label":"Benford Score","data":[0.045,0.032],"borderColor":"rgba(54,162,235,1)","yAxisID":"y"},{"label":"Critical Value","data":[0.028,0.030],"borderColor":"rgba(75,192,75,1)","borderDash":[5,5],"yAxisID":"y2"}],"options":{"scales":{"y":{"position":"left","title":{"display":true,"text":"Benford Score"}},"y2":{"position":"right","title":{"display":true,"text":"Critical Value"},"grid":{"drawOnChartArea":false}}}}}
{{</* /metric_chart */>}}
```

### 4. Volume-Volatility Correlation

```markdown
{{</* metric_chart id="vv-correlation" type="line" title="VV Correlation Over Time" caption="Volume-volatility correlation" */>}}
{"labels":["2023-07-01 00:00","2023-07-01 01:00"],"datasets":[{"label":"VV Correlation","data":[0.85,0.72],"borderColor":"rgba(128,0,128,1)","fill":true,"pointRadius":2,"tension":0.2}]}
{{</* /metric_chart */>}}
```

### 5. Buy/Sell Ratio

```markdown
{{</* metric_chart id="buy-sell-ratio" type="line" title="Buy/Sell Ratio Over Time" */>}}
{"labels":["2023-07-01 00:00","2023-07-01 01:00","2023-07-01 02:00"],"datasets":[{"label":"Buy/Sell Ratio","data":[1.05,0.98,1.12],"borderColor":"rgba(255,99,132,1)"}]}
{{</* /metric_chart */>}}
```

### 6. Time-of-Trade Distribution

```markdown
{{</* metric_chart id="time-of-trade" type="bar" title="Trade Frequency by Second" caption="Trades per second within a minute, aggregated" */>}}
{"labels":["0","1","2","3","4"],"datasets":[{"label":"Trades/s","data":[100,95,110,88,92],"backgroundColor":"rgba(0,128,0,0.7)"},{"label":"Average","type":"line","data":[97,97,97,97,97],"borderColor":"rgba(255,11,11,1)","borderDash":[5,5],"pointRadius":0}]}
{{</* /metric_chart */>}}
```

## Features

| Feature | Detail |
|---------|--------|
| Responsive | Auto-resizes to container, max 900 px wide |
| Dark/light theme | Detects `prefers-color-scheme` and Archie's `dark` CSS class |
| Inline data | JSON embedded at build time — no runtime API calls |
| No new deps | Uses existing `assets/js/chart.js` (Chart.js v4.4.1) |
| Mixed chart types | Combine `bar` + `line` datasets in one chart |
| Dual y-axis | Use `yAxisID` + `options.scales` for two-scale charts |

## How the Pipeline Works

1. Market Health API returns JSON records
2. LLM generates article markdown text
3. `Visualization.generate_report(data, directory)` appends shortcode blocks to `index.md`
4. Hugo renders shortcodes as interactive Chart.js canvases

The `Visualization` class (`tools/python_modules/report_graphics_tool.py`) has
**zero dependencies on matplotlib, pandas, or numpy**.
