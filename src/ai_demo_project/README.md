# AI Product Demo Project

This repository contains an AI product demo project based on the Product Development Kit (PDK) template. The aim of this project is to provide a meaningful demonstration of AI capabilities within the context of distributed network infrastructure and financial health.

## Project Setup
1. Clone the repository.
2. Set up the environment according to the instructions in the 'requirements.txt'.
3. Run the demo using the following command:

   ```bash
   python demo.py
   ```

## Features
- AI-driven market health analysis.
- Real-time data processing and insights.
- Automated decision support for network resilience.
