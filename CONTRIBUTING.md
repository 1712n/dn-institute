## Getting Started

- Winners announced within 7 days of submission review
- Bounties distributed via GitHub Sponsors or direct transfer
- All decisions are final but feedback is provided

## AI Product Challenge Tips 🌰

### Making Your Submission Stand Out 🌰🌰🌰

1. **Originality is key**: Don't just copy the template - transform it!
2. **Use GitHub Models creatively**: Explore different endpoints and use cases
3. **Add personality**: Make your AI product have character
4. **Social integration**: Connect to real social media APIs
5. **Polish matters**: Clean code, good README, deployed demo

### Technical Requirements 🌰

- Must use GitHub Models API
- Must be deployed (GitHub Pages recommended)
- Must include social media integration
- Must have unique value proposition
- Must include 🌰 emojis throughout

### Common Mistakes to Avoid 🌰

- Simply renaming the template without functional changes
- Not deploying the final product
- Missing social media integration
- Generic use cases that don't stand out
