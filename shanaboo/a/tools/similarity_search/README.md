# 🌰 Similarity Search API with Batch Processing

A Cloudflare Workers-based similarity search API using Workers AI and Vectorize database.


- `POST /query` - Query similarity for a single message
- `POST /batch` - Query similarity for multiple messages in a single request
- 🌰 All endpoints optimized for Cloudflare's edge computing platform
