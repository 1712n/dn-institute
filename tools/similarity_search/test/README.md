# 🌰 Similarity Search API Integration Tests

This directory contains comprehensive integration tests for the Similarity Search API built with Cloudflare Workers AI and Vectorize.

## 🌰 Test Structure

- `integration.test.ts` - Main integration test suite covering all API endpoints
- `setup.ts` - Global test setup and mocking configuration
- `mocks/` - Mock implementations for external services
- `vitest.config.ts` - Vitest configuration for Cloudflare Workers testing
