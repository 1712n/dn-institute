## 🌰 Analysis Procedure

### Phase 1: Individual Metric Assessment
Evaluate each metric independently using the thresholds below. Classify each as:
- **NORMAL** — within expected range
- **ELEVATED** — deviating from normal, warrants attention
- **ANOMALOUS** — strong deviation, potential manipulation signal

| Metric | Key | NORMAL | ELEVATED | ANOMALOUS |
|--------|-----|--------|----------|-----------|
| Volume-Volatility Correlation | `vvcorrelation` | ≥ 0.4 | 0.2–0.4 | < 0.2 sustained |
| First-Digit Distribution | `firstdigitdist` | Conforms to Benford's Law | Minor deviations in 1–2 digits | Systematic skew toward high digits (8,9) |
| K-S Test (Benford's Law) | `benfordlawtest` | test ≤ critical value | test slightly > critical | test >> critical value (critical = 1.36 / √tradecount) |
| Volume Distribution | `volumedist` | Power law (skewness > 1, tail exponent < 3) | Reduced skewness or unusual tail | Near-zero/negative skewness; whale clustering without volatility |
| Time-of-Trade | `timeoftrade` | Natural human pattern (irregular) | Slight regularity | Uniform distribution or consistent spikes (bot pattern) |
| Buy/Sell Ratio | `buysellratio`, `buysellratioabs` | 0.4–0.6 with natural variance | Sustained bias (0.3–0.4 or 0.6–0.7) | Extreme bias (<0.3 or >0.7) or unnatural steadiness during volatility |
| VWAP | `vwap` | Price within normal spread of VWAP | Moderate consistent deviation | Large persistent gap between price and VWAP |

### Phase 2: Cross-Metric Correlation
Check for concurrent anomalies that strengthen the manipulation hypothesis:
- **Volume distribution anomaly + low vvcorrelation** → Large trades not moving price = artificial volume
- **First-digit deviation + K-S test failure** → Strong data fabrication signal (e.g., wash trading)
- **VWAP deviation + abnormal buy/sell ratio** → Strategic price manipulation
- **Time-of-trade bot pattern + buy/sell ratio anomaly** → Automated trading system influence
- **Multiple tokens on same exchange showing concurrent anomalies** → Exchange-level issue

### Phase 3: Comparative Context
Where data permits, compare the target exchange against known legitimate exchanges (e.g., Binance, Coinbase) for the same trading pairs. Discrepancies between exchanges strengthen findings.

### Phase 4: Conclusion
- If anomalies are found: rank by severity, identify the entities involved, and describe the likely manipulation mechanism.
- If no significant anomalies are found: state this clearly, cite the metrics that confirm normalcy, and recommend monitoring.
